# Thon Studio Graphic Hooks

Biblioteca privada de sistemas visuais usados como graphic hooks em Reels e videos curtos.

## Uso local

```bash
npm install
npm run capture
npm run dev
```

## Manutencao

Adicione URLs do Instagram em `data/hook-sources.json`.
Rode `npm run capture` para baixar os Reels, capturar screenshots, frames e GIFs.

## Deploy

GitHub Pages usa build estatico do Next.js:

```bash
npm run build
```
